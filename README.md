# Supermarket-sales-annalysis
 Author

Adebajo Ibukunoluwa Ayodele

 Project Overview

This project analyzes supermarket transactional data to generate actionable business insights on revenue performance, customer behavior, and product trends.

Objectives
Identify top-performing product categories
Analyze customer purchasing behavior
Evaluate branch performance
Discover revenue trends over time
Tools & Technologies
Python (Pandas, Matplotlib)
Jupyter Notebook
Dataset

Supermarket sales dataset containing transaction-level data across multiple branches.

Analysis Performed
Data cleaning and preprocessing
Exploratory data analysis (EDA)
Revenue and profit analysis
Customer segmentation
Trend analysis
Key Insights
Food & Beverages is the highest revenue-generating category, indicating strong market demand
Branch A leads in total sales, suggesting a strategic advantage in location or operations
Member customers contribute more revenue than normal customers, highlighting loyalty value
E-wallet and card payments dominate transactions, reflecting digital payment adoption trends
